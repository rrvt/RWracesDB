RacesDB Operations

Toolbar Operations

Open Database File

Opens a database file and displays the number of records in each table.

Member Status Upload Drop Down Command Button

There are three command buttons here but only two commands. Selecting Member Status, the button, will load the Member Status CSV file. Selecting Load CSV File will also load the CSV file. Selecting the "Upload to DB" button will upload any changes in the status to the database. This will result in a description of the changes and attempted changes that have not been made (with the reason for the change not having been made).

Excel CSV File

All current members information will be displayed in the window in a format suitable for loading into Excel. The Save button, , has been configured to save this data with a suitable name (which you are allowed to change).

Google Earth CSV File

The member information needed for Google Earth is displayed on the screen. The Save button, , has been configured to save this data with a suitable name (which you are allowed to change).

Everbridge CSV File

The member information is configured for Everbridge and displayed on the screen. The Save button, , has been configured to save this data with a suitable name (which you are allowed to change).

Responder Counts by Dispatch Category

The counts reflect the rediness of the individuals to report quickly, slowly, maybe and to not at all. This list can be printed and or saved in a text file.

FCC Call Signs of Current Members

Just the call signs sorted. May be printed or saved in a text file.

Member Identifiers CSV File

This list is in CSV format, includes the call sign, name and badge number. It may be printed or saved as a text file.

Badge Expiration Dates, Sorted by Call Sign and by Expiration Date

Three buttons, two commands: Sort list by call sign, Sort list by date. Note members that have earned a badge but one has not been issued yet appear at the top of the list. Due to the pandemic there are several in this category as well as members with badges that have not beeu updated due to the pandemic.

Former Member List Sorted by Name or Call Sign

Three buttons, two commands: Lists former members by name and call sign. Note this list is only a few years old as I started keeping the data in the database and marking the entries as former.

Save File

Displays a Save File Dialog with the type and a filename suggestion based on the data that is displayed in the window and saves the file to the file named.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

Print

Will do all the normal things to print a file.

About

Displays the About Box.

Exit

Exit the application with no reservations.

Menus

File

Refresh

Reload the data from the database into the application.

Save

Displays a Save File Dialog with the type and a filename suggestion based on the data that is displayed in the window and saves the file to the file named.

Print

Will do all the normal things to print a file.

Print Preview

Will display the current report (window) in a print preview window.

Print Setup

Opens the MFC Print Setup dialog box. The orientation is initialized from the Options module.

Make File Copy

Starts an open file dialog box setup to open a database file and then when one is selected copy the content to a file with the same name and " - Copy" appended to it.

Exit

Exit the application with no reservations.

Status Update

Load CSV File

Start an open file dialog box to select the Member Status file downloaded from Google Files. Load the content of the file into the application.

Upload to DB

Upload the Member Status data previously loaded into the application into the database.
